### Features

- **Learners**: Enroll, watch videos, track progress  
- **Instructors**: Create courses, upload videos anytime, earn 80% revenue  
- **Admin**: Full dashboard with monthly revenue chart  
- Instant payment & access (no pending approval)  
- Video + PDF + Link resources  
- Responsive frontend (React + Tailwind)

---

### Tech Stack

**Backend**: Node.js, Express, MongoDB, JWT, Cloudinary  
**Frontend**: React.js, Tailwind CSS, Axios  
**Payment**: Direct bank balance system (20% admin, 80% instructor)
